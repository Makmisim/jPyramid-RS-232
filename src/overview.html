Pyramid Technologies PTalk API - Version 1.0.0.0

This API is designed to support all of our bill validator products. This first release targets the RS-232 protocol and supports a subset of planned features for this interface. The default settings for this API will work with our bill validators and any other RS-232,standard compliant bill validator.
Currently supported commands:

  • Polling: Set the rate at which the slave is polled
  • Escrow Mode: Enable or disable escrow mode
  • Individual Bill Enables: As a master, set the individual bills to accept or reject
  • COM Port Support: All DB9 or USB UART COM ports supported by your OS will work with this API
  • Slave Events: Subscribe to all, some, or none of the slave-generated events
  • Debugging: Logging and debug buffer to help trace issues to their source
  • Extensibility: This API is designed for customization and extensions
